svg-circle&&ellipse

Posted on 2017-07-20 Edited on 2020-03-29 In 前端开发

circle

用于定义圆形
属性：cx,cy,r

解释：cx,cy定义了圆心的位置（相对于左上角）
r定义了圆的半径

ellipse

用于定义椭圆
属性：cx, cy, rx, ry

解释：rx, ry 分别为椭圆两个轴的半轴长
